## 📝 Project Overview
**Digital Life Lessons** is a full-stack MERN application where users can create, store, and share meaningful life lessons and personal growth insights. The platform encourages mindful reflection and community learning through shared knowledge and real-life experiences.

## 🚀 Key Features
* 🔐 **Authentication:** Secure Email/Password and Google login using Firebase Authentication.
* 📝 **Content Management:** Full CRUD functionality to create, update, and delete lessons with public/private visibility.
* 💳 **Premium Membership:** Integrated **Stripe Payment Gateway** for lifetime premium access (৳1500).
* 🌍 **Smart Filtering:** Browse public lessons by category and emotional tone with keyword search.
* ❤️ **User Engagement:** Real-time UI updates for liking, commenting, and saving lessons to favorites.
* 🛠️ **Admin & User Dashboards:** Dedicated management tools for users and an analytics dashboard for admins.

## 🔒 Security & Best Practices
* **Environment Variables:** Credentials are secured using `.env` files.
* **Access Control:** Role-based access control (User & Admin) with Firebase token verification.
* **Data Integrity:** User plans (Free/Premium) are synced directly from MongoDB as the single source of truth.
* **Responsive Design:** Clean, professional UI with consistent alignment, fully optimized for all devices.

## 🚀 How to Run Locally

1. **Clone the repository:**
`git clone https://github.com/sabbir019777/assignment-11-client-side.git`

2. **Install dependencies:**
`npm install`

3. **Start the server:**
`npm run dev`
